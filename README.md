# GigShield-AI

**AI-Powered Parametric Insurance for Gig Workers**

GigShield-AI is a project that provides an AI-driven parametric insurance platform designed to protect gig workers from income loss caused by external disruptions such as heavy traffic, bad weather, or other unpredictable events. It aims to ensure financial stability for gig workers like food delivery partners, ride-share drivers, and freelance service providers.

## Features

- Real-time tracking of gig worker activity
- Parametric insurance claims triggered automatically
- AI algorithms for risk assessment and income loss prediction
- Simple and intuitive interface for gig workers

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub
